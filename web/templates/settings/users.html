Self-Service Authentication

Allow shared users to link their Plex account by visiting a URL. This captures their access token so PlexCache can monitor their OnDeck items. Requires PlexCache to be reachable by the user (reverse proxy, Tailscale, etc.).
Share this URL with your Plex friends. They sign in with their Plex account and their token is saved automatically.

Database Fallback

Path to Plex Media Server's SQLite database file (read-only). When shared users don't have an access token, PlexCache can read their OnDeck items directly from the database. Mount the Plex database directory into the container as a read-only volume, e.g.: - /path/to/plex/Databases:/config/plex-db:ro

Remote User Watchlists

Remote/Friend users can only have their OnDeck monitored directly via API. To cache their Watchlists, provide your Plex RSS URL below.
Generate at: Plex Settings > Watchlist > Generate URL
